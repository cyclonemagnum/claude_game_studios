# Combat-Feel Prototype — v2 (登龍 + 見切 + Boss突刺)

## 现在测试什么
**核心爽感的"完全循环"是否成立**：
读招 → 後跳見切 → 前冲斩 → 大居合 → 登龍

如果这条线打起来爽到上头，肉鸽/关卡/美术都可以围绕这个核心展开。
如果不爽，加再多内容也救不了。

## 关键设计

### 1. 後跳見切 (新)
- 按 U → 0.28秒向后冲, 前 8帧 i-frame
- 命中前 8帧 → 触发見切, 0.55秒窗口内按 J = 前冲斩
- 前冲斩 38伤害 + 加 20气

### 2. 登龍 (新, 终结技)
- 仅在大居合反击命中后 0.6秒内可触发, 消耗 30气
- 三阶段: **戳刺 → 起跳 → 浮空下劈**
- **戳刺(0.32s)** 和 **下劈延迟(0.85s)** 容易被打断
- **起跳(0.18s)** 短暂i-frame
- 命中 = 110伤害 + 大震屏 + 时停
- 被打断 = 0.7秒大破绽

### 3. Boss 新招式: THRUST 突刺 (Phase 2/3)
- 慢起手 (0.95s) + 锁定方向 + 窄锥伤害
- 设计来鼓励玩家用見切应对
- 失败后 1.6 秒大破绽 → 邀请玩家登龍

### 4. 数值微调
- 气衰减 5/s → **2.5/s** (鼓励囤气用大招)
- 大居合失败惩罚 0.7s → **0.4s** (让玩家敢按)
- 大剑伤害 1/2/3x → **1/1.8/3.5x** (Lv3更诱惑)

## 假设验证

### 必看的 3 个体验瞬间
1. **第一次後跳見切成功** — 是否能感受到时停的"刀光石火"？
2. **第一次登龍命中** — 是否觉得这一击改变了战局？
3. **登龍被打断** — 是否觉得"我下次会更小心"而不是"垃圾设计"？

如果 3 个都成立 → 核心爽感存在 → 进入肉鸽/内容扩展
如果 1-2 个不成立 → 调数值 / 改判定
如果都不成立 → 重新审视战斗框架
