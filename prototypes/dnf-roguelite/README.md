# DNF-Roguelite Prototype

## 假设验证
- DNF风格2.5D移动（水平+纵深+跳跃）是否能与怪物猎人武器手感兼容
- 肉鸽关卡循环（杂兵→商店→Boss）是否有正反馈
- 三段斩/崩山裂地斩 技能是否有爽快感

## 核心设计
**DNF + 怪物猎人 + 肉鸽**
- 2.5D视角：X轴水平移动，Y轴纵深移动，空格跳跃
- 3关结构：杂兵房 → 商店 → 杂兵房 → 商店 → Boss
- 商店肉鸽强化：无敌帧/技能范围/冷却缩减/伤害提升

## 操作
| 按键 | 动作 |
|------|------|
| WASD | 移动（W/S为Z轴纵深） |
| Space | 跳跃 |
| J | 普攻（4段连击） |
| K | 三段斩（前冲三连斩，3s CD） |
| L | 崩山裂地斩（跳斩AOE，8s CD） |
| Shift | 闪避（有无敌帧） |
| R | 重开 |

## 技能说明
### 普攻连段 (J)
4段连击，伤害递增：8→10→12→18

### 三段斩 (K) — 参考DNF三段斩
前冲三连斩，每段自动前进，伤害15→20→30。有CD。

### 崩山裂地斩 (L) — 参考DNF狂战士崩山裂地斩
跳起向前下方劈砍，落地时AOE伤害60。有较长CD。
