# DNF-Roguelite Prototype

## 假设验证
- DNF风格2.5D移动 + 怪物猎人武器手感 是否兼容
- 肉鸽关卡循环（杂兵→商店→Boss）是否有正反馈
- 太刀/大剑 + 共通技能(三段斩/崩山裂地斩) 的技能栏设计

## 核心设计: DNF + 怪物猎人 + 肉鸽

### 2.5D 移动
- X轴水平移动，Y轴纵深移动(Z轴)，Space跳跃
- DNF风格：横版卷轴 + 上下纵深 + 跳跃

### 武器系统 (怪物猎人)
- **太刀**: 连段积气 → 居合構え → 小居合/大居合 (见切判定)
- **大剑**: 按住蓄力 → 释放重击，蓄力中闪避 = 肩撞(保留蓄力等级)
- **Tab切换** 两把武器

### 共通技能 (DNF)
- **三段斩** (K): 前冲三连斩，CD制
- **崩山裂地斩** (L/大剑模式): 跳斩AOE，CD制

### 肉鸽循环
- 3关结构: 杂兵房 → 商店 → 杂兵房 → 商店 → Boss
- 商店: 7种buff随机3选1

## 操作

| 按键 | 动作 |
|------|------|
| WASD | 移动（W/S为Z轴纵深） |
| Space | 跳跃 |
| J | 攻击（太刀连段/大剑蓄力） |
| K | 三段斩（共通技能，3s CD） |
| L | 太刀:居合構え(stance中再按=大居合) / 大剑:崩山裂地斩(8s CD) |
| Shift | 闪避(太刀stance中=取消 / 大剑蓄力中=肩撞) |
| Tab | 切换武器 |
| R | 重开 |

## 太刀详细

| 状态 | J | L | Shift |
|------|---|---|-------|
| 通常 | 连段(积气) | 居合構え(需30气) | 闪避 |
| 居合構え中 | 小居合 | 大居合(前冲i-frame) | 取消构え+闪避 |

## 大剑详细

| 状态 | J | L | Shift |
|------|---|---|-------|
| 通常 | 按住蓄力 | 崩山裂地斩 | 闪避 |
| 蓄力中 | 释放=挥砍 | — | 肩撞(保留蓄力) |

## 商店可选buff
🛡️ 延长无敌帧 | ⚔️ 扩大技能范围 | ❤️ 回血 | 🔥 三段斩CD减 | 💥 崩山CD减 | 👊 普攻强化 | 💨 闪避加速
